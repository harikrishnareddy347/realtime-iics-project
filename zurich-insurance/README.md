# Zurich Insurance Project - IICS Practice

This project is designed to provide comprehensive practice materials for learning IICS (Informatica Intelligent Cloud Services) with a focus on:

## Learning Objectives

1. **Mappings** - Basic to advanced data transformation patterns
2. **Mapping Tasks** - Task configuration and execution
3. **Taskflows** - Workflow orchestration and process management
4. **Parameterization** - Dynamic parameter configuration and usage
5. **Error Handling** - Exception handling, retry logic, and recovery
6. **Runtime Options** - Dynamic execution configuration

## Insurance Domain Context

This practice project uses real-world Zurich Insurance scenarios:
- **Customer Data Management** - Customer profile mappings
- **Policy Information** - Policy data transformations
- **Claims Processing** - Claims workflow management
- **Premium Calculations** - Complex data transformations
- **Compliance & Reporting** - Error handling and validation

## Getting Started

1. Start with `documentation/MAPPING_GUIDE.md` for basic mapping concepts
2. Explore `mappings/basic_mappings/` for simple examples
3. Progress to `taskflows/basic_workflows/` for workflow understanding
4. Practice with `parameterization/` examples
5. Study `error_handling/` patterns
6. Implement `runtime_options/` scenarios

## Prerequisites

- Informatica Intelligent Cloud Services (IICS) access
- Basic understanding of ETL concepts
- Familiarity with SQL
- JSON and YAML knowledge
